# 📸 Aesthetica - Image Gallery

## 🔥 CodeAlpha Image Gallery

A responsive image gallery web app where users can upload, view, filter, and delete images with a clean glassmorphic UI.

---

## ✨ Features
- 📂 Upload images  
- 🗂️ Category filtering (Nature, Animals, Architecture, Cities)  
- 🔍 Lightbox with next/previous navigation  
- 🗑️ Delete images  
- 💾 LocalStorage support  
- 📱 Fully responsive  

---

## 🛠️ Tech Stack
HTML, CSS, JavaScript
